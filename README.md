
<h3>Register Form Validation</h3>

<h3> Description </h3>
<h4>
This project is a clean and responsive, designed using standard HTML5, CSS3 and Java Script. It is simple but work according to the logic. In the forms it includes: Username, password, Re-confirm password, E-mail and Register button.
</h4>

<h3>Features </h3>
<ul>
<li>
Centrally aligned, responsive "card" layout.</li>
  <br>
<li>Clean, sans-serif typography for maximum readability.</li>
  <br>
<li> Clearly defined input fields with subtle borders.</li>
  <br>
<li>Descriptive placeholder text for user guidance.</li>
  <br>
<li> Visually distinct blue "Register" action button. </li>
  <br>
<li> Subtle drop-shadow effect for a depth and focus.
</li>
</ul>

<h3> Tech Stack Used</h3>
<ul>
  <li>HTML 5</li>
  <li>CSS 3</li>
  <li>JAVA SCRIPT</li>
</ul>
